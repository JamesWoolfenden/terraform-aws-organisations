Terraform module to manage AWS Organizations.

---

It's 100% Open Source and licensed under the [APACHE2](LICENSE).

## Usage

```cli
$ aws organizations describe-organization
{
    "Organization": {
        "Id": "o-zxxxxxxxx",
        "Arn": "arn:aws:organizations::12312312312:organization/o-zxxxxxxxx",
        "FeatureSet": "CONSOLIDATED_BILLING",
        "MasterAccountArn": "arn:aws:organizations::12312312312:account/o-zxxxxxxxx/12312312312",
        "MasterAccountId": "12312312312",
        "MasterAccountEmail": "anybody@example.com",
        "AvailablePolicyTypes": []
    }
}
```

Include this repository as a module in your existing terraform code:

```hcl
module "organisations" {
  source  = "JamesWoolfenden/organisations/aws"
  version = "0.0.1"
}
```
